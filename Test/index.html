使い方ガイド

  1. Cursorを開き、新しいファイルを作成
  2. テキストやデータを入力・貼り付け
  3. AIに依頼したい箇所を選択し、Ctrl/Cmd + Kでプロンプト入力
  4. サイドバーのチャット機能（Ctrl/Cmd + Shift + L）も活用
  5. AIの提案を受け入れたり、追加指示で調整

非エンジニア向けTips

  • 具体的なプロンプトを心がける
  • AIとの対話を意識する
  • 文脈（目的・読者など）を共有する
  • 試行錯誤を恐れず使ってみる
  • セキュリティ・プライバシーに注意
